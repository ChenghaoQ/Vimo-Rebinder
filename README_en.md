# Vimo Rebinder

Vimo Rebinder is a **desktop application** that helps users implement **custom global shortcuts** to improve daily operations. The app allows users to freely set and manage global shortcuts, supporting **Windows** and **MacOS**. The **Pro Version** is available through the **Microsoft Store** and offers additional features.

## Key Features

* Customizable global shortcuts, supporting **unlimited shortcuts**.
* Supports customizable actions to improve daily operational efficiency.
* Lightweight and efficient, without consuming excessive system resources.
* Supports auto-start, launching the app automatically when the system starts.

## Free vs Pro Version

* **Free Version**: Provides basic global shortcut functionality, suitable for most everyday users.
* **Pro Version**: Offers more advanced features and additional customization options, available through the **Microsoft Store**.

## Installation Instructions

1. Download and extract the Vimo Rebinder zip file.
2. Double-click to run `VimoRebinderSetup.exe` (Windows) or `VimoRebinder.dmg` (MacOS).
3. Follow the installation wizard to complete the setup.
4. Launch the application, configure your settings, and start using custom shortcuts.

## Usage Instructions

1. Launch Vimo Rebinder and go to the settings interface.
2. Click the "Add Shortcut" button to select the keys and actions to bind.
3. After configuring, click "Save" to apply the settings.
4. You can edit or delete any existing shortcuts at any time, and changes will take effect immediately.
